# MCrawler2

## 描述

毛概爬虫第二版，仅限于 bb 平台

## 动机

1. 第一版爬虫用 Angular 构建，当时只有查看答案的功能，但是题目还是得自己做，考虑到题量巨大，会消磨掉许多时间。

2. 于是重写第二版爬虫，试图创建一个通用的命令行工具，与 APP 无关，让无论是 webapp 还是 native app 都能直接调用。

3. 并且增加了自动做题的功能 🚀🚀🚀

## CLI

npm

```
npm install -g mcrawler
```

yarn

```
yarn global add mcrawler
```
